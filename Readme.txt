This is a multi-variate stock price prediction project based on AI.
I have used keras API and tensorflow background. This is my first project on Machine Learning and I got much interest on this.
Functionalities:
This project can predict the next day starting price based on its previous data. I have
used 1 year data of share market and made a model using LSTM(Long Short Term Memory) which test score is satisfying. I have used multiple 
attribute to predict next day opening price.
